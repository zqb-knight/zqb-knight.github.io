PAT 1043 Is It a Binary Search Tree

1. 知识点：

二叉查找树的创建，遍历

2. 题目分析：

要对二叉查找树进行创建，然后判断输入序列是否为前序遍历或镜像前序遍历，最后输出后序遍历

题目易错的地方是，当输入序列是镜像前序遍历时，要输出的是镜像后序遍历。

3. 注意：

3.1 在为二叉树创建节点时要将左右子树初始化为NULL
3.2 涉及到数组的比较时，可以使用vector，可以直接比较数组是否相等，省去了遍历数组的步骤
